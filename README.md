# IDEA_JustCompiling
适用于IDEA的摸鱼插件 | 展示一个"编译中"然后顺理成章地歇一会吧 :)\
[Developers' lives matter](https://996.icu)

## Tips
> 别摸太长时间以免被刀 XD

## How to use
1. 在IDEA里打开一点奇奇怪怪的项目
2. 点击Tools菜单下的"Take a Break"
3. 我摸到鱼辣.jpg
4. 不想摸了点取消

## How to INSTALL!
1. Download jar from release
2. Google for "how to install jetbrains plugin from disk"
3. Now you got it :)
